11. IMPROVEMENT OPPORTUNITIES
================================================================================

What Should Be Improved (Priority Order):

  1. ACCESSIBILITY -- Color contrast (gold on dark passes WCAG AA only at
     larger sizes), focus indicators, skip-to-content link, semantic HTML

  2. REAL VISUALS -- Replace stock photography with real product photos,
     team photos, office/warehouse images

  3. SOCIAL PROOF -- Add client logos, testimonials with real names/photos,
     case studies, export volume stats

  4. FORM UX -- Floating labels, inline validation, multi-step RFQ with
     progress indicator, country code phone input

  5. PERFORMANCE -- Minify CSS/JS, lazy-load below-fold images, cache fonts,
     add resource hints, consider WebP for images

  6. MICRO-INTERACTIONS -- Scroll-triggered counters, hover states on all
     interactive elements, loading states on forms

  7. HERO RELIABILITY -- Replace video with high-quality image carousel as
     primary, video as enhancement

  8. NAVIGATION -- Add mega-menu for product categories, sticky CTA on mobile,
     search auto-suggest

  9. FOOTER -- Add newsletter signup, active social media links, partner
     certifications as badges

  10. COLOR SYSTEM -- Consider adding a subtle green accent for agricultural/
      fresh product categories

What Should Remain Unchanged:
  - Gold + Navy color system (distinctive and appropriate)
  - Poppins + Inter typography pairing
  - CSS variable architecture
  - RTL support system
  - RFQ cart functionality
  - Section structure (page header -> content -> footer)
  - Button design language

Visual Problems:
  - Stock photography undermines authenticity
  - No visual identity beyond colors (no pattern, texture, illustration style)
  - Hero video has no controls, autoplays data on mobile
  - Product card images have inconsistent quality
  - SVG-only favicon (no PNG fallback for older browsers)

Opportunities:
  - Introduce custom icon set or illustration style for feature cards
  - Add animated data visualizations for market reach
  - Create visual asset library (product photography style guide, iconography)
  - Add video testimonials or virtual tour of facilities
  - Implement dark/light mode toggle
  - Add a "Trust Bar" with recognizable client logos
